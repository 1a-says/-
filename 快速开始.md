# 快速开始指南

## 📦 安装依赖

由于您系统的npm存在权限问题，请尝试以下任一方法：

### 方法1：右键以管理员身份运行 `install.bat`
这是最简单的方法，双击 `install.bat` 文件即可自动安装。

### 方法2：手动命令安装
打开PowerShell（以管理员身份），执行：
```powershell
cd "d:\图书馆管理系统"
npm install
```

### 方法3：使用其他包管理器
如果npm一直有问题，可以安装yarn：
```powershell
npm install -g yarn
cd "d:\图书馆管理系统"
yarn
```

## 🚀 启动项目

### 方法1：双击 `start.bat`
安装完依赖后，直接双击 `start.bat` 启动开发服务器。

### 方法2：手动命令启动
```powershell
cd "d:\图书馆管理系统"
npm run dev
```

项目会自动在浏览器中打开 `http://localhost:3000`

## ❓ 常见问题

### Q: npm install 失败怎么办？
A: 
1. 以管理员身份运行
2. 关闭杀毒软件
3. 使用 yarn 替代 npm

### Q: 页面空白无法显示？
A: 检查浏览器控制台是否有错误，确保所有依赖都已正确安装

### Q: 如何添加新的测试账号？
A: 编辑 `src/views/Login.vue` 文件中的 `mockUsers` 对象

### Q: 密码锁定如何解除？
A: 等待30分钟，或者刷新页面（前端模拟）
